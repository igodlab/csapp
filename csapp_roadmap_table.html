Non-associativity directly explains why parallel reductions give different results at different thread counts.

This is the most important chapter in the book for GPU work. Memory coalescing in CUDA is cache-line awareness applied to GPU DRAM bursts. Shared memory tiling is manual cache management. The memory mountain experiment is the CPU version of the roofline model you'll use constantly. Bank conflicts in shared memory are conflict misses in disguise. Everything in PMPP Ch. 6–9 will make more sense after deeply understanding this chapter.

Signal handling matters when building robust GPU training jobs: SIGTERM/SIGINT handling for checkpoint-on-interrupt requires async-signal-safe code. Also relevant for understanding why CUDA context creation is per-process and what happens to GPU state when a process crashes.
